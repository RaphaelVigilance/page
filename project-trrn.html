TRRN Railway Challenge 2026

  • Stack: ATmega2560 · ESP32 · MQTT · Node-RED · AWS Cloud · C++
  • Role: Hardware / Firmware / Control System
  • Event: Thailand Railway Challenge 2025–2026

ระบบควบคุมรถรางอัจฉริยะที่สามารถสั่งการขับเคลื่อนได้ในระยะไกล พร้อมระบบตรวจสอบสถานะการทำงานแบบ Real-time พัฒนาขึ้นสำหรับการแข่งขัน TRRN Railway Challenge

Project Overview

ออกแบบและพัฒนาระบบควบคุมรถรางสำหรับการแข่งขัน TRRN Railway Challenge โดยบูรณาการ Microcontroller เข้ากับระบบ IoT เพื่อให้ผู้ควบคุมสามารถสั่งงานและ ติดตามสถานะรถรางได้แบบ Real-time ผ่านเครือข่ายอินเทอร์เน็ต ระบบรองรับการส่งคำสั่งขับเคลื่อนทั้งแบบ Manual และ Scheduled รวมถึง การแจ้งเตือนเมื่อระบบผิดปกติ

System Architecture

  • สั่งการขับเคลื่อนในระยะไกลผ่าน IoT Dashboard
  • ตรวจสอบสถานะแบบ Real-time บน Node-RED
  • ATmega2560 จัดการ Logic ควบคุมหลัก
  • ESP32 เชื่อมต่อกับ AWS Cloud ผ่าน MQTT
  • ระบบแจ้งเตือนเมื่อตรวจพบความผิดปกติ
  • ออกแบบวงจรไฟฟ้าและ Wiring ครบถ้วน

Technical Stack

  • ATmega2560
  • ESP32
  • MQTT Protocol
  • Node-RED
  • AWS Cloud
  • C++
  • Electrical Design & Wiring
