รายละเอียดโปรเจกต์

ระบบ IOT โดยมี Node-RED เป็น Server และสื่อสารผ่าน MQTT Protocol เข้า ESP32 สื่อสารกับ ATmega328P ด้วย UART Protocol

การทำงานของระบบ

  • Node-RED รับคำสั่งผ่าน Dashboard
  • ส่งข้อมูลผ่าน MQTT Broker
  • ESP32 รับ MQTT แล้วส่งต่อผ่าน UART
  • ATmega328P ควบคุมอุปกรณ์ปลายทาง
