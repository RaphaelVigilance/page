Handmade Microcontroller

  • Stack: ATmega328P · PCB Design · C++
  • Role: Hardware / Firmware
  • Status: Deployed in production projects

ออกแบบและสร้าง Microcontroller Board ตั้งแต่ต้นจนจบ — ตั้งแต่วาด Schematic ออกแบบ PCB Layout บัดกรีชิ้นส่วน จนถึงเขียน Firmware และนำไปใช้งานจริงในโปรเจกต์

Project Overview

แทนที่จะใช้บอร์ดสำเร็จรูปอย่าง Arduino โปรเจกต์นี้มุ่งเน้นการสร้าง Microcontroller Board ขึ้นมาเอง โดยใช้ ATmega328P เป็นหัวใจหลัก กระบวนการครอบคลุมตั้งแต่การออกแบบวงจร Schematic การวาง PCB Layout การบัดกรีชิ้นส่วนทั้งหมดด้วยมือ และการเขียน Firmware ด้วย C++ บอร์ดที่ได้ถูกนำไปใช้งานจริงในโปรเจกต์อื่น ๆ ต่อไป

Development Process

  • ออกแบบวงจร Schematic
  • วาง PCB Layout และ Route สาย
  • บัดกรีและประกอบชิ้นส่วนทั้งหมด
  • เขียน Firmware ด้วย C
  • ทดสอบ Functionality และ Debug
  • Deploy ใช้งานจริงในโปรเจกต์

Technical Stack

  • ATmega328P
  • C
  • PCB Design
  • Schematic Design
  • Soldering
  • Electronic Circuit Design
