Smart Home IoT System

  • Stack: Node-RED · MQTT · ESP32 · ATmega328P · C++
  • Role: Hardware / Firmware / Server
  • Status: Completed

ระบบบ้านอัจฉริยะที่สามารถควบคุมและตรวจสอบอุปกรณ์ภายในบ้านได้แบบ Real-time ผ่าน Dashboard บนเว็บเบราว์เซอร์หรือสมาร์ทโฟน

Project Overview

โปรเจกต์นี้พัฒนาระบบ IoT สำหรับควบคุมอุปกรณ์ภายในบ้าน โดยใช้ Node-RED เป็น Server หลักสำหรับประมวลผล Logic และแสดง Dashboard ผู้ใช้สามารถสั่งงานผ่านหน้าเว็บได้ทันที ข้อมูลคำสั่งถูกส่งผ่าน MQTT Protocol ไปยัง ESP32 ซึ่งทำหน้าที่เป็น Gateway สื่อสารต่อไปยัง ATmega328P ด้วย UART เพื่อควบคุมอุปกรณ์ปลายทาง

System Architecture

  • Node-RED รับคำสั่งและแสดง Dashboard
  • MQTT Broker เป็นตัวกลางรับ-ส่งข้อมูล
  • ESP32 รับ MQTT และแปลงเป็น UART
  • ATmega328P ควบคุมอุปกรณ์ปลายทาง
  • Real-time Feedback จากอุปกรณ์กลับสู่ Dashboard

Technical Stack

  • Node-RED
  • MQTT Protocol
  • ESP32
  • ATmega328P
  • UART Protocol
  • C++
